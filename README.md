# ChatingRoom

Java实现的多人聊天系统

### 项目流程
首先启动server端.之后可以任意起一些客户端

### Done

- 群聊功能

- 实时显示最新的在线用户列表

- 用户启动客户端后会随机产生一个昵称,之后用户可以修改自己的id并实时更新在所有用户的在线用户列表和聊天记录中
